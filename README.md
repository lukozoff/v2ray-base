# 🚀 V2Ray Configs Ultimate (v1.4.1)

Автоматизированный репозиторий с проверенными конфигурациями. В отличие от обычных списков, здесь находятся только те прокси, которые прошли **«боевую» проверку**.

---

### 🔍 Как это работает?
* **Сбор:** Скрипт собирает более **40,000** конфигов из проверенных источников и локальных файлов.
* **Фильтрация:** Мы не просто пингуем порт. Скрипт запускает каждый конфиг через ядро `xray.exe` и делает реальный запрос к Gstatic (`generate_204`).
* **Результат:** В файл `V2rayCompletetest.txt` попадают только те сервера, которые реально пропускают интернет-трафик.
* **Сортировка:** Все конфиги отсортированы по задержке (**Real Delay**) — самые быстрые в начале списка.

Конфиги берутся из 60 источников (на 09.03.2026)
