🚀 V2Ray Configs Ultimate (v1.4.1)
Автоматизированный репозиторий с проверенными конфигурациями. В отличие от обычных списков, здесь находятся только те прокси, которые прошли «боевую» проверку.

🔍 Как это работает?
Сбор: Скрипт собирает более 18,000 ссылок из проверенных источников и локальных файлов.

Фильтрация: Мы не просто пингуем порт. Скрипт запускает каждый конфиг через ядро xray.exe и делает реальный запрос к Google (generate_204).

Результат: В файл V2rayCompletetest.txt попадают только те сервера, которые реально пропускают интернет-трафик.

Сортировка: Все конфиги отсортированы по задержке (Real Delay) — самые быстрые в начале списка.

📥 Как пользоваться
Просто добавь ссылку на этот файл в свой клиент (v2rayN, v2rayNG и др.) как подписку:
https://raw.githubusercontent.com/lukozoff/v2ray-base/main/V2rayCompletetest.txt
